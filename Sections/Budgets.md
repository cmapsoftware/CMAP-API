# Budgets
Use budgets to return budget specific data for a client or project

## Get Externals
* `* GET v1/budgets/1/externals` Returns the additionals/externals for a given project

```javascript
[{ 
	"id" : "16",
	"name" : "Additional Fees"	
},{ 
	"id" : "17",
	"name" : "Reimbursable Travel Expenses"	
}]
```
